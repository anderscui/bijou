## 新词发现

使用 `bijous.extractors.oov`：

```python
file = '/data/comments.txt'

TOP = 10000
MAX_LEN = 5
MIN_FREQ = 10
with_score = False

extracted = extract_words(file, TOP, max_len=MAX_LEN, min_count=MIN_FREQ, with_score=with_score)
print(len(extracted))

for k, score in extracted:
    if is_in_jieba(k):
        continue
    print(k, score)
```
